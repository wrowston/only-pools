Pool Owners can create a new Survivor or Confidence Pool from scratch or reuse the setup of a prior-season Pool. Creation is available only when the service has an Available Season with at least one valid Start Week.

## Create from scratch

1. Open **My Pools**.
2. Choose **Create Pool**.
3. Under **Setup**, keep **New setup** selected.
4. Enter a Pool name.
5. Choose the **Pool Type**: Survivor or Confidence.
6. Choose the **Start Week**.
7. Choose a **Pick Lock mode**.
8. Choose **Create Pool**.

The Pool Season and Pool type are permanent for that Pool. The Start Week becomes immutable when the Pool accepts its first pick.

## Choose a Pick Lock mode

The mode applies to everyone in the Pool:

- **Game Kickoff** locks a Survivor Pick when its selected game starts. In Confidence, each prediction and its confidence value lock independently when that game starts; later games remain editable.
- **Weekly Cutoff** still locks games that start early, then locks every remaining pick, confidence assignment, and Weekly Tiebreaker Prediction Sunday at 1:00 p.m. Eastern.

A schedule change can move a lock that has not happened yet. It never reopens a Pick Lock that has already been reached.

<GuideCallout>
The Pool Ruleset becomes read-only at the earlier of the first server-accepted competitive edit or the first included game reaching a Pick Lock. Review the type, Start Week, and lock mode before people begin picking.
</GuideCallout>

## Create from a template

A Pool Template reuses setup without carrying over competitive history or enrolling anyone automatically.

1. In **Create Pool**, select **From template**.
2. Choose a prior Pool.
3. Review the prefilled name, type, Start Week, and Pick Lock mode.
4. For each former Participant, choose **Member**, **Admin**, or **Skip**.
5. Create the Pool.

If you selected former Participants, Only Pools displays a person-specific Returning Participant Invite for each one. Copy and send each link to the named person. Nobody joins until that person opens and accepts their own invite.

An Admin choice is only a proposed role. The person must explicitly accept the Returning Participant Invite before receiving Pool Admin authority.

## If Create Pool is disabled

The button remains disabled until an Available Season exists. You can still accept a valid Pool Invite and join someone else’s Pool. Refresh **My Pools** later to check whether creation has become available.

## After creation

Open the Pool’s **Pool** section and choose **Create / retrieve invite**. This requires Step-up Verification because the invite is a reusable credential. Continue with [Invite People and Join a Pool](/guides/invites-and-joining).
