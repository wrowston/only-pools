In a Confidence Pool, predict the winner of every Required Confidence Game and assign each prediction a unique confidence value. Correct predictions earn the assigned value; incorrect or missing predictions earn zero.

## Pick winners

1. Open the Confidence Pool and choose **Board**.
2. For each game, choose the team you predict will win.
3. Wait for **Saved** after each change.

Every game on the frozen Pick Sheet is required. Pool Owners cannot omit matchups. You may change an unlocked prediction by choosing the other team.

## Assign confidence values

Each required game already has one unique value from the Confidence Scale. Use the **Conf** selector beside an unlocked game to move a value.

When you assign a value that another unlocked game currently holds, Only Pools swaps the values so the set remains unique. Values attached to locked games are no longer available for reassignment.

The scale is consistent across the season. When a week has fewer NFL games, its lowest values are left out. For example, if the season maximum is 16 and the current Pick Sheet has 14 games, the available values are 3 through 16.

## Weekly Tiebreaker Prediction

Enter a whole number from 0 through 200 for the combined final points in the chronologically last scheduled Pick Sheet game. The designated game does not change after the Pick Sheet freezes.

The prediction ranks Participants who have the same weekly points. Smaller absolute error ranks ahead; when two predictions are equally far away, the lower prediction ranks ahead. A valid prediction ranks ahead of an omitted one. The tiebreaker adds no points and does not break a Season Standing tie.

The field locks with its designated game under Game Kickoff mode or at the Weekly Cutoff.

## Participant-started and incomplete sets

Your Confidence Pick Set becomes Participant-started when the server accepts at least one winner prediction, confidence-value change, or Weekly Tiebreaker Prediction. Merely opening the Board or changing something locally without a successful save does not start it.

If a started set reaches its locks with missing winner predictions, each missing prediction earns zero. Valid locked predictions still score normally, and you remain in the Weekly Standing.

## Automatic picks for an untouched week

If the first Required Confidence Game locks while your set is still completely untouched, Only Pools creates an Automatic Confidence Pick Set. It predicts every home team and uses the Default Confidence Ranking.

Later games that remain unlocked can still be edited, and their available confidence values can still be rearranged. The set continues to show its automatic origin so the fallback is clear.

<GuideCallout>
An automatic set protects a completely untouched week. It does not fill blanks in a Participant-started set; those missing locked predictions earn zero.
</GuideCallout>

## Locks and canceled games

Under Game Kickoff mode, each prediction and its assigned value lock independently when that game starts. Under Weekly Cutoff mode, early games lock individually and everything remaining locks Sunday at 1:00 p.m. Eastern.

A canceled Required Confidence Game remains on the frozen Pick Sheet and awards zero points. Its slot is not removed or replaced.

Open **Standings** to compare weekly points, possible remaining points, and cumulative Season Standings.
