# 12 — Reuse a prior Pool as a template

**What to build:** A Pool Owner creates a new Pool from a prior-season Pool Template that prefills name, type, ruleset, lock mode, and valid Start Week preference without copying memberships, picks, standings, or audit history. Optional Returning Participant Invites let prior people opt in; nobody is auto-enrolled; only the Owner may propose Admin via those invites.

**Blocked by:** 03 — Create an Active Pool and open the Week Board; 11 — Administer roles, removal, archive, and audit
